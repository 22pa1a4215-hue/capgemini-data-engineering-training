# Associate Data Engineering – Learning Pathway

## Overview  
This repository captures my journey through the Data Engineer Learning Plan, focusing on building strong fundamentals in data ingestion, pipeline development, and deployment using modern data engineering tools and best practices.

---

## Modules Covered  

### 🔹 Data Ingestion with Lakeflow Connect
Learned techniques to ingest data efficiently from multiple sources into Lakeflow, ensuring reliability and scalability.

---

### 🔹 Deploy Workloads with Lakeflow Jobs
Explored scheduling, automation, and monitoring of workloads using Lakeflow Jobs for production-ready pipelines.

---

### 🔹 Build Data Pipelines with Lakeflow Spark Declarative Pipelines
Designed and implemented Spark pipelines declaratively, transforming raw data into structured, usable formats.

---

### 🔹 DevOps Essentials for Data Engineering
Acquired skills in version control, CI/CD, and monitoring for robust and maintainable data pipelines.
